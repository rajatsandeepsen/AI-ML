# Module 3: Data Preprocessing & Visualization

This module covers how to work with real-world datasets before model training.

## Learning Goals

- Understand dataset structure and data types
- Detect and handle missing values
- Clean and transform data for analysis
- Perform basic feature operations
- Create clear visualizations using Matplotlib and Seaborn

## Folder Structure

- `datasets/` → sample datasets for practice
- `01_data_understanding.ipynb` → dataset inspection basics
- `02_data_cleaning.ipynb` → missing values and transformations
- `03_visualization_basics.ipynb` → charts with Matplotlib/Seaborn

## Suggested Flow

1. Open `01_data_understanding.ipynb`
2. Continue with `02_data_cleaning.ipynb`
3. Finish with `03_visualization_basics.ipynb`

## Prerequisites

- Python basics (Module 2)
- Basic NumPy and Pandas knowledge

## Expected Outcome

By the end of this module, you should be able to prepare raw data for ML workflows and produce useful charts for exploratory analysis.
